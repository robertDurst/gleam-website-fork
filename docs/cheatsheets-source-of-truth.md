# Cheatsheets - A Source of Truth

Definition of Cheatsheet:
> a focused overview of Gleam, articulated through structured comparison to another programming language.

Kicked off by [this Github discussion](https://github.com/gleam-lang/gleam/discussions/4981), this document aims to standardize the feature set, style, and overall content of the [Gleam website's cheatsheets](https://gleam.run/documentation/#cheatsheets).
